## 📋 О проекте

**IDEF3 Editor** — это десктопное приложение для создания и редактирования диаграмм в нотации **IDEF3** (Integration DEFinition for Process Description Capture Method). Программа предназначена для моделирования бизнес-процессов, сценариев и последовательности выполнения работ.

Проект разрабатывается в рамках выпускной квалификационной работы по направлению «Информационные системы и технологии».

### 🎯 Цель разработки

Создание доступного, легковесного и специализированного инструмента для работы с IDEF3-диаграммами, который:
- не требует дорогостоящего лицензирования;
- обеспечивает проверку синтаксической корректности диаграмм;
- имеет интуитивно понятный интерфейс;
- не перегружен избыточным функционалом.

## 🚀 Возможности

### ✅ Реализовано
- [x] Создание новой диаграммы
- [x] Сохранение диаграммы в JSON-файл
- [x] Загрузка ранее сохраненной диаграммы
- [x] Добавление UOB-блоков (Unit of Behavior)
- [x] Добавление перекрестков (Junction):
  - `&` — логическое «И»
  - `X` — исключающее «ИЛИ»
  - `O` — логическое «ИЛИ»
- [x] Создание связей между элементами
- [x] Перемещение элементов по холсту
- [x] Редактирование текстовых меток
- [x] Удаление элементов
- [x] Экспорт диаграммы в PNG
- [x] Валидация диаграмм по правилам IDEF3
- [x] Undo/Redo (отмена/повтор действий)
- [x] Масштабирование рабочей области
